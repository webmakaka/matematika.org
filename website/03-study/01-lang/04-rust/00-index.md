Оригинал 2019

**GitHub:**  
https://github.com/rust-lang/book/

<br/>

```
$ cd ~/tmp/
$ curl https://sh.rustup.rs -sSf | sh

$ rustup update

$ source $HOME/.cargo/env
$ rustc --version
$ cargo --version
```

<br/>

```
$ cargo new hello_cargo
$ cd hello_cargo/
$ cargo build
$ cargo run
$ cargo check
$ cargo build --release
```
