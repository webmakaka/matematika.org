### Сокращения, которые часто используют

```
Org1, Org2, Org3, Org4: Organizations
C1, C2: Channels
Pr1, Pr2, Pr3: Peers
O4: Orderer
L1, L2: Ledgers
NC4: Network configuration
CC1: Channel configuration
CA1, CA2, CA3, CA4: Certificate authorities

S1 - smart contract
A1 - client/application
```
